# 欢迎

**wfCMS 系统内容管理 fastify , art-template ,webpack 同构 **

## 主要技术

* fastify + rethinkdb + nats-streaming + docker 后端 *
* fastify + art-template + webpack 前端  *

## 开发
    
- 拉取代码

    `git clone https://github.com/hppking/wfCMS.git`

- 前端页面开发环境:
    `cd wfCMS`
    `yarn install`
    `npm run dev`

- 前端页面生产环境:
    `npm run build`
    `npm run prod`

- 后台系统管理页面开发:
    `cd manage`
    `npm run dev`
